# VOID_RUNNER - Foundational Mandate & Roadmap

## 1. Project Overview
VOID_RUNNER is a high-fidelity "hacker-game" prototype built with Angular 19. It focuses on the strategic acquisition of real-world hardware appliances to optimize manual hacking operations (mini-games).

## 2. Core Mechanics
- **Hardware Synergy:** Hardware items like the WiFi Pineapple or Rubber Ducky provide passive and active bonuses.
- **Manual Hacking:** Tactical mini-games: Port Scanning, Brute Forcing, SQL Injection, and RFID Cloning.
- **Global Routing:** Utilize VPNs and the Onion network to bounce signals across the globe, increasing stealth at the cost of credits and speed.
- **Detection System:** Real-time IDS (Intrusion Detection System) that traces your activity. Stealth hardware and secure routing are critical for survival.

## 3. Real-World Inspiration
- **Reconnaissance:** Simulated port scanning (inspired by `nmap`).
- **Exploitation:** Brute-force attacks (`hydra`) and SQL Injection (`sqlmap`).
- **Hardware:** Hak5 gear (Pineapple, Ducky) and physical research tools (Proxmark3).
- **Networking:** Multi-node routing mimicking Tor (Onion) and secure tunnels (VPN).

## 4. Technical Stack
- **Framework:** Angular 19+ (Standalone, Zoneless).
- **State:** Angular Signals (Core state, computed bonuses, network logic).
- **3D Visualization:** Three.js for a real-time wireframe cyber-globe.
- **Audio:** Web Audio API for immersive feedback.
